About

About me:

Mathematician. I have been working as a Java Web Developer for a while. Currently based in Barcelona.

About this site:

This site is a static website build on [Flask](https://flask.palletsprojects.com/en/1.1.x/) with [Flask-Flatpages](https://flask-flatpages.readthedocs.io/en/v0.7.1/) and [Frozen-Flask](https://pythonhosted.org/Frozen-Flask/). The styling is completely based on the Jekyll's theme [Tale](https://github.com/chesterhow/tale) created by [ChesterHow](https://github.com/chesterhow). Go ahead and take a look at [his website](https://chester.how/). I have planned to modify it in the future in order to have something more personal, but for sure is a very nice foundation to work with.
