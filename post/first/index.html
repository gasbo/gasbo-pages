My First Entry

Starting this space of random thoughts before the year is over. It's always better to be already done with your new year resolutions before the year even starts. I'll try to post here whatever development I am currently working in. Maybe I will post some of my old ones too, I don't know.

I am currently working as a Java Web Developer so I won't be talking much about that. Surely you can expect to see things about WebAssembly, Numerical Analysis, Data Science (not anything related to NN tho), some Python snipets and a bit of basic C. Furthermore, I might bring some interesting mathy topics around if I find the time to do so!

This site is a static website build on Flask with Flask-Flatpages and Frozen-Flask. The styling is completely based on the Jekyll's theme Tale created by ChesterHow. Go ahead and take a look at his website. I have planned to modify it in the future in order to have something more personal, but for sure is a very nice foundation to work with.

See you later!
